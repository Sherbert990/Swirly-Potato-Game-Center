# Don't Look Down

A browser-based platformer game inspired by Gimkit's "Don't Look Down" mode. Climb as high as you can without falling!

## How to Play

Open `dont_look_down.html` in any web browser — no installation needed.

### Controls

| Action | Keyboard | Mobile |
|--------|----------|--------|
| Move left | `←` or `A` | ◀ button |
| Move right | `→` or `D` | ▶ button |
| Jump | `↑`, `W`, or `Space` | ▲ Jump button |
| Double jump | Press jump again mid-air | Tap jump again mid-air |

## Features

- Infinite procedurally generated platforms
- Double jump — press jump a second time while airborne
- Collectable stars scattered across platforms
- Height tracker to measure your progress
- Mobile-friendly on-screen controls
- Blue-themed visual style

## Running the Game

No server or dependencies needed. Just double-click `dont_look_down.html` or drag it into your browser.

## Built With

- HTML5 Canvas
- Vanilla JavaScript
- No external libraries
